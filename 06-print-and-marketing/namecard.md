# Namecard Guide

## Physical Specifications

- **Size:** 90 × 54mm (standard business card)
- **Bleed:** 3mm on all sides (production file: 96 × 60mm)
- **Safe zone:** 5mm inside the trim line (keep all text within 5mm from the edge)
- **Resolution:** 300dpi minimum
- **Colour mode:** CMYK
- **Stock:** 400gsm minimum
- **Finish:** Soft-touch matte laminate
- **Optional:** Gold foil on the CM underrule line (back of card only)

---

## Front of Card

**Background:** Deep Navy (#071E3D)

### Elements (top to bottom, left to right):

**Top-left — CM Mark:**
- White variant, 20mm height
- Positioned 5mm from top edge, 5mm from left edge

**Top-right corner accent:**
- 3px × 3px gold corner mark (#C8921A)
- Positioned flush with the top-right corner (inside bleed)

**Centre-left block (name and title):**

- Name: Cormorant Garamond 400, 11pt, White
- Gold hairline rule (1px, #C8921A): 18mm wide, 3pt below the baseline of the name
- "DIRECTOR" (or job title): Sora 300, 6pt, ALL CAPS, letter-spacing 0.14em, White, 4pt below the rule
- "OPERATIONS" (or department): Sora 300, 5.5pt, ALL CAPS, letter-spacing 0.12em, Text Light (#6B849C), 2pt below title

**Bottom-left — Entity name:**
- Sora 300, 6pt, ALL CAPS, letter-spacing 0.10em, Text Light (#6B849C)
- Positioned 5mm from bottom edge

### Director Variant (Ivan Ignatius Ang):
```
CM MARK (white, 20mm)                         [gold corner accent]

IVAN IGNATIUS ANG
────────────────── (18mm gold rule)
DIRECTOR
OPERATIONS


CITI MAJU GROUP
```

## Back of Card

**Background:** Warm Paper (#F5F2EC)

### Layout: Two columns separated by a vertical hairline

**Left column (~35mm wide):**
- CM Mark (navy variant), 14mm height
- Centred in the column, vertically centred on the card

**Vertical divider:**
- 1px hairline, full card height (minus 5mm top and bottom safe zone)
- Colour: #C8921A (gold)
- Positioned at the column break point

**Right column (contact details):**

All contact labels in Sora 600, 5pt, ALL CAPS, letter-spacing 0.10em, Text Mid colour
All contact values in Cormorant Garamond 400, 8.5pt, Text Dark colour

```
M   +6012-379 6562
E   inquiry@citimaju.com
W   www.citimaju.com
A   [Address Line 1]
    [Address Line 2]
    [Postcode City]
    [State]
```

**Tagline (optional, bottom of right column):**
- "TRUSTED. CERTIFIED. DELIVERED." in Sora 300, 6pt, ALL CAPS, letter-spacing 0.14em, Text Light

## Ordering Instructions

When ordering from a print vendor:

1. Provide the PDF print file (with bleeds and crop marks)
2. Specify: 400gsm, soft-touch matte laminate, 90 × 54mm trim
3. Request a press proof before full print run — verify gold (#C8921A) matches Pantone 117 C
4. If ordering gold foil: specify the CM underrule on the back as the foil zone; provide a separate foil mask file
5. Minimum order: 100 cards per variant (soft-touch laminate is not economical below this)

```

```

CMG-Namecard-Group-IvanIgnatiusAng-Back.pdf
