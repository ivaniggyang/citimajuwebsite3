## Quick Reference

Full colour specifications are in `01-brand-foundation/colours.md`. This is a quick reference only.

| Name | Hex | Use |
|---|---|---|
| Deep Navy | `#071E3D` | Dark backgrounds, namecard front |
| Brand Blue | `#1B4F8A` | CM mark, UI elements, primary headings |
| Blue Mid | `#0D3362` | Secondary dark blue |
| Blue Light | `#2E6BB5` | Hover states |
| Blue Pale | `#E8F1FB` | Subtle backgrounds, badges |
| **Refined Gold** | **`#C8921A`** | **Logo rule, hairlines — accent only** |
| Gold Light | `#DBA830` | Hover on gold |
| Gold Dark | `#A07010` | Deep gold, pressed state |
| Warm Paper | `#F5F2EC` | Letterhead, document backgrounds |
| Cream | `#F7F4EF` | Alternate backgrounds |
| White | `#FFFFFF` | Base, reversed text |
| Text Dark | `#0D1B2E` | Primary body text |
| Text Mid | `#3A5068` | Secondary text |
| Text Light | `#6B849C` | Muted text, captions |

## Critical Note

**Gold is `#C8921A`. Never use `#E8A020`** — that is the retired amber colour, replaced in the current brand refresh. If you see `#E8A020` in any file, replace it immediately.
