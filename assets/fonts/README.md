# Font Assets

## Brand Typefaces

Citi Maju uses three typefaces. All are available via Google Fonts at no cost under the Open Font License (OFL), which permits free use in commercial projects.

---

## 1. Cormorant Garamond

**Role:** Display headings (H1, H2), logo mark
**Weights used:** 400 (Regular), 500 (Medium), 600 (SemiBold), and 400 Italic

**Google Fonts page:**
https://fonts.google.com/specimen/Cormorant+Garamond

**Download (zip, all weights):**
https://fonts.google.com/download?family=Cormorant+Garamond

**Licence:** SIL Open Font License 1.1 — free for commercial use, modification, and distribution with attribution.

**Designer:** Christian Thalmann

---

## Web Use — Single Import URL

For web projects, import all three fonts in a single call:

```html
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,400;0,500;0,600;1,400&family=Sora:wght@300;400;500;600;700;800&family=Noto+Sans:wght@300;400;500;600&display=swap" rel="stylesheet">
```

Or as CSS `@import`:
```css
@import url('https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,400;0,500;0,600;1,400&family=Sora:wght@300;400;500;600;700;800&family=Noto+Sans:wght@300;400;500;600&display=swap');
```

## Desktop Installation (Windows)

1. Download the zip files from the Google Fonts links above
2. Extract the zip
3. Select all `.ttf` files
4. Right-click → Install for all users
5. Restart any open Office or design applications

## Desktop Installation (macOS)

1. Download and extract the zip files
2. Double-click each `.ttf` file → click "Install Font"
3. Or open Font Book → drag the `.ttf` files in
4. Restart any open applications

---

## Microsoft Office Fallbacks

When the brand fonts are not available on a user's system:

| Brand Font | Office Fallback |
|---|---|
| Cormorant Garamond | Garamond (built-in on Windows) |
| Sora | Gill Sans MT or Trebuchet MS |
| Noto Sans | Calibri or Arial |

Install the correct fonts on all machines that produce Citi Maju documents. Fallbacks are for emergencies only.
