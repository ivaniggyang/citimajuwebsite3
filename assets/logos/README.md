# Logo Assets

## Files in This Directory

| File | Description | Use On |
|---|---|---|
| `cm-mark-blue.svg` | CM mark, Brand Blue (#1B4F8A) letters, gold rule | White or light backgrounds |
| `cm-mark-white.svg` | CM mark, White letters, gold rule | Navy or dark backgrounds |
| `cm-mark-navy.svg` | CM mark, Deep Navy (#071E3D) letters, gold rule | Warm Paper or Cream backgrounds |

## Usage Notes

- The gold underrule (`#C8921A`, 1.5px) is fixed on all variants. Never change it.
- These SVGs use a web-safe serif fallback stack. For print, the font must be embedded. Open in Illustrator or Inkscape, expand the text to outlines, and export as EPS or PDF for print use.
- Minimum display size: 32px height (screen) / 12mm height (print)
- All SVGs are transparent background — place on your desired background colour

## Cormorant Garamond Requirement

These SVGs reference Cormorant Garamond. If viewed in a browser without the font installed, the fallback (Garamond or serif) will render. For correct rendering:
- Embed the Google Fonts link in the page: see `assets/fonts/README.md`
- Or open in a design tool with Cormorant Garamond installed

## File Naming Convention

```
cm-mark-[colour].svg             — Mark only
cm-wordmark-[entity].svg         — Wordmark only (to be added)
cm-lockup-h-[entity]-[bg].svg   — Horizontal lockup (to be added)
cm-lockup-s-[entity]-[bg].svg   — Stacked lockup (to be added)
```

Wordmark and lockup SVGs will be added once the complete entity variants are finalised.
