# Letterhead Guide

## Overview

The Citi Maju letterhead is used for all formal correspondence: letters, site instructions, notices, and formal confirmations. There are two entity variants — Engineering and Construction — and one Group variant for general correspondence not tied to a specific entity.

---

## Layout Specifications

### Page Setup
- **Paper:** A4 (210 × 297mm)
- **Background:** Warm Paper (#F5F2EC) or White (#FFFFFF)
- **Margins:** Top 32mm, Bottom 24mm, Left 25mm, Right 25mm
- **Print resolution:** 300dpi minimum

### Header Zone (Top 28mm)
The header occupies the full page width from the top edge.

**Left column (logo block):**
- CM Mark lockup (horizontal): height 16mm
- Entity name in Sora 300, ALL CAPS, 7pt, letter-spacing 0.14em, Text Mid colour
- Placed at the top-left margin

**Right column (company details):**
- Entity registered name: Sora 600, 7pt, Text Dark
- Registration number: Sora 300, 6pt, Text Mid
- Address: Sora 300, 6.5pt, Text Mid, right-aligned
- CIDB / SPAN / Air Selangor cert line (Engineering only): Sora 300, 6pt, Text Light, right-aligned, below address

**Divider:**
- 1px hairline rule the full page width, colour #C8921A (Refined Gold), below the header block
- Gap: 4mm below the rule before body text begins

### Footer Zone (Bottom 20mm)
Centred, 1px gold hairline rule across the full width above the footer text.

Footer text: `inquiry@citimaju.com  ·  www.citimaju.com  ·  Klang Valley, Malaysia`
- Sora 300, 6.5pt, ALL CAPS, letter-spacing 0.1em, Text Light colour

Page number (if multi-page): right-aligned in footer, Sora 300, 7pt, Text Light

## Entity Variants

### Group Letterhead
- Logo: CM Mark + "CITI MAJU GROUP" wordmark
- Registration block: omit (group is not a single registered entity)
- Certifications: omit

### Engineering Letterhead
- Logo: CM Mark + "CITI MAJU ENGINEERING SDN BHD" wordmark
- Registration: Company No. [insert reg no.], Sora 300
- Certifications (right column): CIDB G7 Grade · SPAN Registered · Air Selangor Approved Contractor

### Construction Letterhead
- Logo: CM Mark + "CITI MAJU CONSTRUCTION SDN BHD" wordmark
- Registration: Company No. [insert reg no.], Sora 300
- Certifications: omit

## Typography on Letterhead

| Element | Font | Size |
|---|---|---|
| Letter date | Sora 400 | 9pt |
| Reference line | Sora 500, ALL CAPS | 8pt |
| Recipient address block | Noto Sans 400 | 9pt |
| Salutation | Noto Sans 400 | 10pt |
| Body text | Noto Sans 400 | 9.5–10pt |
| Subject line | Noto Sans 600 | 10pt |
| Sign-off | Noto Sans 400 | 10pt |
| Signatory name | Cormorant Garamond 400 | 13pt |
| Signatory title | Sora 300, ALL CAPS | 7pt |

## Letter Structure

```
[Date — right-aligned or left-aligned, Sora, 9pt]

[Recipient Name]
[Designation]
[Company Name]
[Address Line 1]
[Address Line 2]
[Postcode City]

Dear [Name / Sir or Madam],

Re: [Subject of letter in bold]

[Body paragraphs — Noto Sans 400, 10pt, line height 1.65, 6pt space between paragraphs]

Yours sincerely / Yours faithfully,

[Signature space — 20mm]

[FULL NAME IN CORMORANT GARAMOND]
[TITLE — SORA 300, ALL CAPS, 7PT]
[Entity name]
```

## File Formats

Letterhead is available as:
- **Word template (.dotx):** For internal document drafting
- **InDesign template (.indd):** For high-quality print output
- **PDF shell:** For adding text in Acrobat without layout software

Contact the design lead for template files. Do not recreate the letterhead from scratch — use the approved template.

---

## Print Specifications

For physical letterhead paper stock:
- Paper weight: 120gsm minimum
- Finish: uncoated or lightly coated
- Print: offset lithography preferred; digital print acceptable for small runs
- CMYK conversion: verify gold (#C8921A) against press proof — target Pantone 117 C
