# Site Instructions Guide

## What Is a Site Instruction

A Site Instruction (SI) is a formal written directive issued by the project superintendent, engineer, or contract administrator to the contractor, or by the Citi Maju project manager to site supervisors and subcontractors.

SIs document changes to scope, method, or programme. They are not informal WhatsApp instructions. Any verbal instruction that affects scope, cost, or programme must be followed up with a formal SI.

---

## When to Issue a Site Instruction

Issue an SI when:
- A change to the original scope of works is directed
- A different construction method is required
- A health and safety requirement changes the approach
- Client or engineer directions are given on site
- A Variation Order may result from the instruction

---

## Site Instruction Reference

`[Entity]-SI-[Project Code]-[SI No.]`

Example: `CME-SI-SA01-012`

Number SIs consecutively per project. Do not restart numbering.

---

## Required Fields

1. **SI reference number**
2. **Date of issue**
3. **Project name and contract reference**
4. **Issued by** — name, designation, and signature
5. **Issued to** — name and designation of the recipient
6. **Description of instruction** — what is to be done, clearly and completely
7. **Affected area / chainage / location** — be specific
8. **Effect on programme** — state if the SI affects the construction timeline
9. **Commercial impact** — state "No commercial impact" OR "Variation Order to follow" — do not leave this blank
10. **Acknowledgement block** — space for the recipient to sign and date, confirming receipt

---

## Writing the Instruction

The instruction body must be unambiguous. Anyone on site reading it must be able to execute it without calling you for clarification.

**Structure the instruction body as:**
1. The background / context (one sentence if needed)
2. The instruction itself — what is to be done, using imperative verbs
3. The location or scope boundary
4. Any constraints (timing, materials, method)

**Good example:**
> Extend the 200mm HDPE pipeline from MH07 to MH08A (approximately 42m), following the alignment shown on the attached sketch. Works to commence after reinstatement of the existing trench at MH07. Material: Class PN12.5, matching the existing specification. Pressure test to be conducted before backfill.
>
> Commercial impact: Variation Order to follow. Do not proceed without written authorisation if cumulative variation value exceeds the approved limit.

**Bad example:**
> Please extend the pipe further as discussed.

## Design and Format

SIs are issued on the entity letterhead.

**Title block:** "SITE INSTRUCTION" in Sora 700, 12pt, ALL CAPS, Brand Blue, below the letterhead header.

**Reference and date:** Sora 500, 9pt, Text Dark, in a two-column table below the title block.

**Instruction body:** Noto Sans 400, 10pt, line height 1.65.

**Acknowledgement section:** Table at the bottom — "Issued by" on the left, "Received by" on the right. Include name, designation, signature, and date fields for both.

**Attachments:** List any sketches, drawings, or specifications attached.

---

## Distribution

Original: Retained by the issuing party
Copy 1: Issued to the recipient (site supervisor or subcontractor)
Copy 2: Project file

If issued on site: print, sign, hand to the recipient. Request their signature on the original. Photograph the signed copy before leaving the site.

If issued remotely: issue as a signed PDF. Request acknowledgement by return email or WhatsApp, then follow up for the countersigned copy.
