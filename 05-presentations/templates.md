# Presentation Templates

## Available Templates

The following PowerPoint templates are maintained in the company's shared drive. Always start from a template — do not build a presentation from scratch.

| Template | Use Case | File Name |
|---|---|---|
| Company Introduction | General company overview for new client meetings | `CMG-Template-Intro.pptx` |
| Tender / Bid Presentation | Formal bid presentations for tenders | `CMG-Template-Tender.pptx` |
| Project Progress Update | Monthly or milestone progress reporting | `CMG-Template-Progress.pptx` |
| Executive Briefing | Short briefings for senior client or GLC stakeholders | `CMG-Template-Executive.pptx` |

## Fonts in Templates

Templates embed the required fonts. If you open a template and see font substitution warnings, install the fonts first:

1. Download from `assets/fonts/` or from the Google Fonts link in the typography guide
2. Install on your system (double-click the .ttf file → Install)
3. Close and reopen PowerPoint
4. Open the template again — fonts should now render correctly

## Editing a Template Safely

**Do:**
- Duplicate a slide from the master to create new slides — do not insert blank slides and reformat from scratch
- Replace placeholder text (the grey "Click to edit" fields)
- Update the date and reference fields in the footer/cover
- Add your own photography to the designated image placeholder zones

**Do not:**
- Move or resize the logo on any slide master
- Change font styles or sizes on existing layouts
- Alter the colour of any text boxes that use theme colours
- Delete slides from the slide master view

If you need a layout that does not exist in the template, contact the design lead rather than improvising a custom layout.

## Exporting for Delivery

**For screen / email delivery:**
- File → Export → Create PDF/XPS
- Resolution: Standard (optimised for screen)
- File name: `CitiMaju-[Type]-[ClientOrProject]-[YYYY-MM-DD].pdf`

**For printing:**
- File → Export → Create PDF/XPS
- Resolution: High quality print
- Ensure slide size matches the print format

**For sharing editable files (internal only):**
- Save as `.pptx`
- Do not share editable PowerPoint files with external parties

## Template Requests

If your presentation requires a layout or configuration not covered by existing templates, raise the request with the design lead. Do not adapt or create ad-hoc designs outside the brand system — inconsistent presentations in circulation undermine the brand.
