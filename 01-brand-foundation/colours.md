# Colour System

## Palette Overview

The Citi Maju colour palette is built around steel blue and refined gold. Navy carries weight and authority. Gold is used sparingly — it is an accent, not a theme.

---

## Primary Colours

### Deep Navy
- **Hex:** `#071E3D`
- **RGB:** 7, 30, 61
- **CMYK:** 89, 51, 0, 76 (approx)
- **Use:** Primary dark backgrounds, namecard front face, headers on dark-mode documents, site signage backgrounds.

### Brand Blue
- **Hex:** `#1B4F8A`
- **RGB:** 27, 79, 138
- **CMYK:** 80, 43, 0, 46 (approx)
- **Use:** Primary mark colour (CM initials), primary UI elements, section headers on light backgrounds.

### Blue Mid
- **Hex:** `#0D3362`
- **RGB:** 13, 51, 98
- **CMYK:** 87, 48, 0, 62 (approx)
- **Use:** Secondary dark blue. Document headers, sidebar panels, deep background layers.

## Gold Accent

### Refined Gold *(Primary)*
- **Hex:** `#C8921A`
- **RGB:** 200, 146, 26
- **CMYK:** 0, 27, 87, 22 (approx)
- **Use:** Logo underrule, hairlines, accent borders, key highlights. **Never for large fills.**

> **Critical:** This is `#C8921A`. Never use `#E8A020`. The old amber is retired. If you see `#E8A020` anywhere, replace it.

### Gold Light
- **Hex:** `#DBA830`
- **RGB:** 219, 168, 48
- **CMYK:** 0, 23, 78, 14 (approx)
- **Use:** Hover state on gold elements only.

### Gold Dark
- **Hex:** `#A07010`
- **RGB:** 160, 112, 16
- **CMYK:** 0, 30, 90, 37 (approx)
- **Use:** Pressed/active state on gold elements. Deep gold for high-contrast applications.

## Colour Usage Rules

**Gold is an accent, not a theme.** The gold in any material should cover no more than 5–10% of the visual area. If you are filling a panel, box, or background in gold, you are using it wrong.

**Navy is the dominant dark.** When a dark background is needed, use `#071E3D`. Do not use black (`#000000`) for backgrounds.

**Never combine the blue mark on a blue background.** Ensure at minimum a 3:1 contrast ratio between the mark and its background.

**Text colours on dark backgrounds:** Use white (`#FFFFFF`) for primary text. Use Blue Pale (`#E8F1FB`) for secondary text. Do not use Text Dark, Text Mid, or Text Light on dark backgrounds — they will not meet contrast requirements.

**Print note:** When specifying CMYK values for offset print, request a press proof. Screen representations of `#C8921A` vary; verify the gold against a Pantone reference — the closest Pantone equivalent is approximately **Pantone 117 C**.
